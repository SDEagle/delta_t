# DeltaT

DeltaT provides an interface to represent time differences and make calculations with them nice and easy.

## Installation

Add this line to your application's Gemfile:

    gem 'delta_t'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install delta_t
